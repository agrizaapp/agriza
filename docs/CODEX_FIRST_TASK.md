# Primeira tarefa sugerida para o Codex

Faça uma auditoria sem alterar comportamento:

1. catalogue todos os campos de data e locais de exibição;
2. catalogue todos os tipos de lançamento;
3. identifique quais já possuem editar, excluir e cancelar;
4. identifique quais gravações ainda ocorrem sem confirmação;
5. produza um plano de mudanças por módulo;
6. proponha testes de regressão;
7. só então implemente em commits pequenos.

Não mude o esquema do PostgreSQL nessa primeira tarefa.
